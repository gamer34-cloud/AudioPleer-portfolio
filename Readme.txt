1. Установть все зависимости команндой npm i в корне проекта и в папке express-backend
2. Для того чтобы запустить бэкенд следует перейти в папку express-backend командой: "cd express-backend"
3. Когда мы перешли в express-backend нам следует запустить его командой npm run start
4. Теперь надо собрать проект, для начала надо выйти из папки express-backend и из корня проекта написать комманду npm run build или же npx vite build
5. Теперь мы можем ввести последнюю команду под названием npm run dev или же npx vite
6. Тепеь перейдите на хост который написал вам vite!
